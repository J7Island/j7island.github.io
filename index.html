Recent Projects

  1. Model Checking SmartThings SmartApps
    • Summary: detect smart home apps' vulnerabilities.
    • Transform the Groovy source code of Samsung SmartThings SmartApps to PSharp code.
    • Use PSharp to model check the transformed code.
    • Use the generated .dgml le to verify the communication channel of the SmartApps.

  2. Vigilia: Securing Smart Home Edge Computing
    • Summary: prevent mailicious communications in smart home apps.
    • Provide a programming model (Set and Relation) for developing applications.
    • Instrument the application code at the phase of the installation.
    • Generate the routing policy based on the con gurations users generated.
    • Block the malicious communication by the generated routing policy.

  3. PerfBlower: Quickly Detecting Memory-Related Performance Problems via Amplication
    • Summary: detect Java applications’ performance bugs.
    • Use the Instrumentation Speci cation Language to describe and detect memory related performance bugs.
    • Use C to build a data structure invariants detector.
    • Hack the JVM and use JVM tool interface (JVMTI) to instrument the application code, for detecting the allocation sites that cause data structure invariants.
